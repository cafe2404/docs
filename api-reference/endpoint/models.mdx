## Get Models

Retrieve a list of all available text-to-speech models.

<RequestExample>
```bash
curl -X GET "https://api.ttslab.com/api/models/" \
  -H "X-API-KEY: your_api_key_here"
```

```python
import requests

url = "https://api.ttslab.com/api/models/"
headers = {
    "X-API-KEY": "your_api_key_here"
}

response = requests.get(url, headers=headers)
print(response.json())
```

```javascript
const fetch = require('node-fetch');

const url = 'https://api.ttslab.com/api/models/';
const headers = {
    'X-API-KEY': 'your_api_key_here'
};

fetch(url, { headers })
    .then(response => response.json())
    .then(data => console.log(data));
```
</RequestExample>

### Model Properties

| Property | Type | Description |
|----------|------|-------------|
| `id` | string | Unique model identifier |
| `name` | string | Human-readable model name |
| `description` | string | Model description |
| `version` | string | Model version |
| `languages` | array | Supported language codes |
| `features` | array | Model features |
| `max_text_length` | integer | Maximum text length in characters |
| `credits_per_second` | float | Credits consumed per second of audio |

### Language Codes

| Code | Language |
|------|----------|
| `en` | English |
| `es` | Spanish |
| `fr` | French |
| `de` | German |
| `it` | Italian |
| `pt` | Portuguese |
| `pl` | Polish |
| `hi` | Hindi |
| `ja` | Japanese |
| `ko` | Korean |
| `zh` | Chinese |

### Features

| Feature | Description |
|---------|-------------|
| `multilingual` | Supports multiple languages |
| `high_quality` | Optimized for quality over speed |
| `fast` | Optimized for speed over quality |
| `efficient` | Low resource consumption |
| `balanced` | Good balance of quality and speed |
