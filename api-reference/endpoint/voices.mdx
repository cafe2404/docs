## Get Voices

Retrieve a list of all available voices.

<RequestExample>
```bash
curl -X GET "https://api.ttslab.com/api/voices/" \
  -H "X-API-KEY: your_api_key_here"
```

```python
import requests

url = "https://api.ttslab.com/api/voices/"
headers = {
    "X-API-KEY": "your_api_key_here"
}

response = requests.get(url, headers=headers)
print(response.json())
```

```javascript
const fetch = require('node-fetch');

const url = 'https://api.ttslab.com/api/voices/';
const headers = {
    'X-API-KEY': 'your_api_key_here'
};

fetch(url, { headers })
    .then(response => response.json())
```

## Create Voice

Create a new custom voice from audio samples.

<RequestExample>
```bash
curl -X POST "https://api.ttslab.com/api/voices/" \
  -H "X-API-KEY: your_api_key_here" \
  -H "Content-Type: multipart/form-data" \
  -F "name=My Custom Voice" \
  -F "description=A custom voice for my project" \
  -F "files=@sample1.mp3" \
  -F "files=@sample2.mp3"
```

```python
import requests

url = "https://api.ttslab.com/api/voices/"
headers = {
    "X-API-KEY": "your_api_key_here"
}

data = {
    "name": "My Custom Voice",
    "description": "A custom voice for my project"
}

files = [
    ("files", ("sample1.mp3", open("sample1.mp3", "rb"), "audio/mpeg")),
    ("files", ("sample2.mp3", open("sample2.mp3", "rb"), "audio/mpeg"))
]

response = requests.post(url, headers=headers, data=data, files=files)
print(response.json())
```
</RequestExample>

### Request Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `name` | string | Yes | Voice name |
| `description` | string | No | Voice description |
| `files` | file[] | Yes | Audio sample files (MP3, WAV, M4A) |

## Update Voice

Update voice settings and metadata.

<RequestExample>
```bash
curl -X PUT "https://api.ttslab.com/api/voices/JBFqnCBsd6RMkjVDRZzb" \
  -H "X-API-KEY: your_api_key_here" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Updated Voice Name",
    "description": "Updated description",
    "settings": {
      "stability": 0.7,
      "similarity_boost": 0.8
    }
  }'
```

```python
import requests

voice_id = "JBFqnCBsd6RMkjVDRZzb"
url = f"https://api.ttslab.com/api/voices/{voice_id}"
headers = {
    "X-API-KEY": "your_api_key_here",
    "Content-Type": "application/json"
}

data = {
    "name": "Updated Voice Name",
    "description": "Updated description",
    "settings": {
        "stability": 0.7,
        "similarity_boost": 0.8
    }
}

response = requests.put(url, headers=headers, json=data)
print(response.json())
```
</RequestExample>

## Delete Voice

Delete a custom voice.

<RequestExample>
```bash
curl -X DELETE "https://api.ttslab.com/api/voices/custom_voice_id_123" \
  -H "X-API-KEY: your_api_key_here"
```

```python
import requests

voice_id = "custom_voice_id_123"
url = f"https://api.ttslab.com/api/voices/{voice_id}"
headers = {
    "X-API-KEY": "your_api_key_here"
}

response = requests.delete(url, headers=headers)
print(response.json())
```
</RequestExample>

### Voice Properties

| Property | Type | Description |
|----------|------|-------------|
| `id` | string | Unique voice identifier |
| `name` | string | Voice name |
| `description` | string | Voice description |
| `category` | string | Voice category (professional, casual, custom) |
| `language` | string | Primary language code |
| `gender` | string | Voice gender (male, female, auto) |
| `age` | string | Voice age group (young, adult, senior, auto) |
| `accent` | string | Voice accent (american, british, australian, etc.) |
| `sample_url` | string | URL to voice sample audio |
| `created_at` | string | Creation timestamp |
| `is_public` | boolean | Whether voice is publicly available |
| `status` | string | Voice status (active, processing, failed) |
| `settings` | object | Voice generation settings |

### Voice Settings

| Setting | Type | Range | Description |
|---------|------|-------|-------------|
| `stability` | float | 0.0 - 1.0 | Controls voice consistency |
| `similarity_boost` | float | 0.0 - 1.0 | Controls voice similarity to original |
| `style` | float | 0.0 - 1.0 | Controls speaking style |
| `use_speaker_boost` | boolean | - | Enhances speaker clarity |
